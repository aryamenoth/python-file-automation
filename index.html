The Problem

Manual file sorting is a massive bottleneck. Inconsistent naming, mixed formats, and human error lead to lost productivity and data chaos.

  • 15 mins per batch
  • High error rate

The Objective

Deploy a Python-based script to automate directory scanning, file type identification, and logical sorting into dynamic folders.

  • Instant execution
  • 100% Accuracy
System Architecture

Automated Pipeline

01

Input Directory

User selects the source folder containing unstructured files.

02

OS Scan & Parse

System iterates through files using `os.listdir()` and identifies extensions.

03

Dynamic Logic

Checks if category folders exist. If not, creates them via `os.mkdir()`.

04

Execution (Move)

`shutil.move()` relocates files to their respective directories instantly.
